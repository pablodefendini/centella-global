About these inserts

Two 4×7 slip-in sheets that live inside the program brochure's pockets. Same paper, ink, type, and color system as the program and pre-packet — sized to fit, pulled out for use, returned to the pocket.

Front pocket
Texting contacts — Prime Movers (24) and Next Generation Leaders (16). Names and phone numbers only, in two cohort columns. Built for in-the-moment coordination during the convening.
Back pocket
Overview agenda — the detailed schedule from pre-packet page 5, recast for the back-pocket take-with-you sheet. Same plenary (amber) and breakout (orange) color-coding.
Format
4×7 inches portrait, one side, full-bleed. All phone numbers are placeholders (NANP fictional-use 555-01XX range) until the real cohort directory is in.
Print
Browser print dialog renders to 4in × 7in via the @page rule. Production PDFs are built by npm run prime-movers:pdf alongside program.pdf and pre-packet.pdf.
